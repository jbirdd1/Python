Problem 3: For Loops

Python provides the for loop that can be used to simplify programs that contain iteration. It is a simpler way to use iteration than using while loops.
The general form of a for statement is:
for variable in sequence:
code block
Given the string "Python Is A Great Programming Language!", write Python code using for loops and if statements to print out the number of vowels and constants in the string.

Problem 4: Creating Lists

A list is an ordered sequence of values, where each value is identified by an index. A list can contain values of any type. An empty list is written as [].
For example, the code,
myList = [1, 2, 3]
print myList
produces the output,
[1, 2, 3]
Create a list containing the values 1, 2.2, 3, 4.2. Then write Python code using a loop, that prints out each element in the list and prints out the average value of all of the elements within the list.

Problem 5: List Appending

Python provides methods that operate on lists. For example, append will add a new item to the end of a list.
For example, the code,
list1 = ["a", "b", "c"]
list2 = ["d", "e"]
list3 = list1 + list2

list1.append(list2)
print list3
print list1
produces the output,
['a', 'b', 'c', 'd', 'e']
['a', 'b', 'c', ['d', 'e']]
